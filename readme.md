## Proyecto 1

**Problema**: Una tarde me contraba con mis tres mejores amigas, nos encontrabamos platicando cuando derrepente se nos acerca una ni√±a y nos pregunta si nos gusta BTS, le respondemos que si y nos empieza a hacer burla diciendo que ella es la que sabe todo de BTS, nos empezo a retar como una total Hate-ARMY. No nos podiamos quedar calladas y decir que era cierto, por que no lo era, solo queria hacernos molestar, entonces las tres hicimos un circulo y pensamos como podriamos resolverlo.

**Solucion:** A las tres se nos prendio el foco y decidimos hacer una *Trivia* para ver si sabia mas que nosotras.

**Recreacion:** Las cuantro pensamos en preguntas y decidimos crear un *Diagrama* con varias preguntas para despues hacerlo.
